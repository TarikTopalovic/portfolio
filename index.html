Hayaku Pro

BUILD 01 / LOCAL AI PIPELINE

// excuse for building: learning Japanese so I never have to read subtitles againDrop in any YouTube video or audio file → get a word-by-word annotated Japanese transcript. Four AI models chained on local GPU — speech recognition, speaker diarization, morphological analysis, neural translation — zero paid APIs. Hover any word: reading + meaning. Every word feeds a frequency-ranked vocab bank with one-click Anki export.

  • 4 ML models, one orchestrated pipeline

  • GPU fallback chain: fp16 → int8 → CPU
  • graceful degradation at every layer

UniPlan Pro

BUILD 02 / CONSTRAINT SOLVER

// excuse for building: picking classes by hand once was already once too manyUniversity publishes timetables as garbage PDFs — times like 1500-17::50, multi-course rows, broken tables. UniPlan parses the mess, then enumerates every valid lecture + tutorial combination, counts conflicts, and renders the best schedules on a hand-drawn canvas. One click → .ics with recurrence rules, straight into your phone calendar. Shipped as a double-click .exe to classmates who've never seen a terminal.

  • regex parser hardened on real dirty data
  • combinatorial search, best-first ranking
  • custom canvas timetable renderer
  • .ics export with timezone + RRULE
dirty PDF→ pdfplumber + regex→ course catalog→ cartesian search→ conflict ranking→ .ics calendar

OLX Sniper

Macro Studio

BUILD 04 / INPUT AUTOMATION

// excuse for building: clicking the same buttons every day is a job for the computerDesktop macro engine that records every mouse move and keystroke, then replays them with DirectInput-level injection — adjustable speed, infinite loops, global hotkeys. A quick-actions panel covers the classics: auto-clicker, key spammer, type-this-text, launch-that-app. A humanizer layer adds micro-delays and pixel jitter so the playback doesn't feel robotic.
